    • Why all these maintenance releases of legacy code? Mostly because they are still useful and/or important to me (and hopefully others too). Some points to consider beyond saving them from bit-rot:

    • GWOCSS is the most flexible (and only GPL'd) windfield model currently available
    • CCCC is still the best open source complexity metrics tool available for common languages
    • URT is still the primary RLE library for run-length-encoded images (ie, netpbm depends on it for RLE support)
    • Cyclo is another useful metrics tool that works in cases where the cccc parser rejects too much code
